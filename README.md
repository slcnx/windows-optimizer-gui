# Windows Optimizer GUI (高阶性能优化与精细控制台)

> **专为高级用户、开发者与极客设计的高阶 Windows 性能控制工具，兼具进程 CPU 核心亲和性绑核监控与系统服务/定时任务单项多选精细控制功能。基于原生 PowerShell 打造，支持编译为单文件免安装 EXE，干净、轻量、无后门！**

---

## 💡 为什么需要这个工具？（设计初衷与核心痛点解决）

市面上虽然存在大量的“系统优化神器”，但对于高级开发者和硬核玩家来说，它们普遍存在以下痛点，也是本项目要彻底打破的：

### 1. 拒绝“一键优化”的黑盒误伤，把控制权归还给用户
绝大多数传统优化软件都采用粗暴的 **“一键优化”** 和 **“一键还原”** 模式。这种一刀切的做法极易误伤核心业务：
- 例如盲目关闭了打印机服务 (`Spooler`)、Windows 搜索与索引 (`WSearch`)、诊断服务 (`DiagTrack`) 或各类后台计划任务，最终导致系统某些基础应用异常报错、打印机找不到或搜索框假死。
- 当出现问题后，传统工具只提供“全部一键还原”，无法让用户在不改变其他优化的前提下，对单个冲突的服务进行开关。
- **本工具解决方案**：彻底废弃“一键优化/还原”。我们提供了一个交互式的多选/单选精细化清单控制台，清晰展示当前每一个优化项（服务与计划任务）的实时运行状态，支持用户使用鼠标单选或按住 `Ctrl` / `Shift` 进行多选，按需“开启/恢复”或“停止/禁用”任意一项，完全掌控自己的操作系统！

### 2. 攻克大小核 CPU 亲和性调度限制与长效锁定痛点
现代多核处理器（尤其是大小核 P-core/E-core 混合架构）在面对高负载渲染或高并发游戏时，系统默认线程调度可能导致核心频繁切换或抢占，进而引发画面卡顿与延迟尖峰。
- 在 Windows 自带的任务管理器中，即便手动为某个进程设置了亲和性（绑核），一旦该进程重启，或者该程序通过启动器（Launcher）重新拉起了新的衍生/子进程，绑核设置便会瞬间丢失，用户不得不每次启动都手动重复配置，极其繁琐。
- **本工具解决方案**：不仅能清晰展示系统中高资源进程当前绑定的 CPU 核心亲和掩码（Mask）并实现秒级切换，更首创支持 **“自动追踪并锁定绑核设置”** 功能！后台低消耗定时器会以微秒级感知程序状态，只要发现选定绑核的目标程序重启或拉起了衍生子进程，立即自动将预设的核心亲和性应用至新进程，彻底解决重开失效的难题。

### 3. 轻量化单文件可执行程序 (`.exe`) 零配置部署
我们深知极客用户对“臃肿客户端和常驻后台全家桶”的厌恶：
- **原生技术栈**：本项目使用系统原生的 PowerShell 脚本驱动底层 Windows Management/System API，并采用纯代码创建现代化的 `WinForms` 图形界面（双标签页流畅设计）。
- **打包为单文件可执行程序**：随仓库发行的 [WindowsOptimizerGUI.exe](https://github.com/slcnx/windows-optimizer-gui/releases) 为完全嵌入脚本内核的独立单文件程序，删去了控制台黑框（ `-noConsole` ），并在程序清单中内置了管理员权限提权配置（ `-requireAdmin` ）。双击即用，无需配置环境，无任何垃圾文件占用。

## 🌟 核心功能一览

### Tab 1：进程绑核监控与长效锁定
- 📋 **高负载进程监控**：自动排序并清晰展示内存或 CPU 占用较高的核心进程。
- ⚙️ **精准核心绑定**：支持按需勾选指定的物理/逻辑核心（例如 `Core 0` ~ `Core 3`），点击【应用绑核】立即更改内核亲和性位掩码 (`ProcessorAffinity`)。
- 🔄 **自动追踪衍生进程**：勾选“自动追踪并锁定绑核设置”后，无论目标程序启动了多少子进程或频繁重新启动，工具都会自动且长效地将对应核心偏好锁定到底。

### Tab 2：系统服务与计划任务单项/多选精细控制
- 💡 **多维度清单**：覆盖并监控诸如 `SysMain` (超级预读)、`Windows Search`、`Spooler` 打印池、`DiagTrack` 遥测服务、游戏网络 API 以及各类微软诊断与用户体验改善计划任务。
- 🎯 **自由圈选控制**：支持通过 `Ctrl` 和 `Shift` 自由框选任意数量的项目，精准执行 **【✅ 开启 / 恢复所选项】** 与 **【⛔ 停止 / 禁用所选项】**。
- 🌈 **直观状态反显**：根据服务实际运行状态与启动类型，实时以鲜明绿色（开启）与红色/灰色（停用）进行视觉反馈。

## 🚀 如何使用（开箱即用 EXE 版本）

1. 访问本项目的右侧 **[Releases (发行版)](https://github.com/slcnx/windows-optimizer-gui/releases)** 页面。
2. 下载最新的可执行文件：`WindowsOptimizerGUI.exe`。
3. 在任意 Windows 10 / Windows 11 电脑上双击该文件（系统会提示 UAC 管理员提权，请点击“是”）。
4. 启动后即可在两个功能标签页之间自由切换体验！

## 🛠️ 源代码与二次定制

如果您是开发者并希望查阅、修改或二次打包本工具：

1. 克隆本项目源代码：
   ```powershell
   git clone https://github.com/slcnx/windows-optimizer-gui.git
   cd windows-optimizer-gui
   ```
2. 直接在管理员权限的 PowerShell 下运行脚本进行开发调试：
   ```powershell
   .\WindowsOptimizerGUI.ps1
   ```
3. 使用 `PS2EXE` 将修改后的 `.ps1` 重新打包为独立 `.exe`：
   ```powershell
   Install-Module ps2exe -Scope CurrentUser -Force
   Invoke-ps2exe -InputFile .\WindowsOptimizerGUI.ps1 -OutputFile .\WindowsOptimizerGUI.exe -noConsole -requireAdmin -title "Windows Optimizer GUI" -description "精细控制与性能优化工具" -version "2.0.0.0"
   ```
